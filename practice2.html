if文の作成

if文を使って年齢によって表示を変える処理を作りましょう

まず変数ageを使用して自分の年齢を代入してください

次に年齢が18歳以上の場合には「成人」、18歳未満の場合は「未成年」とコンソールに表示されるif文を作成しましょう

if文の構文

if (条件式) {
条件が真の場合の処理
} else {
条件が偽の場合の処理
}

今回の場合は…

if (18歳以上であることを判定する条件式) {
「成人」とコンソールに表示する処理
} else {
「未成年」と表示する処理
}

最後にコンソールを確認してみましょう！

ブラウザ上で右クリック→検証→コンソールでコンソールが開けます
